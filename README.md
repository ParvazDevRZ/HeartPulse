# HeartPulse

HeartPulse is a particle-based heart animation developed using **C++20**, **SFML 2.6**, and **GLSL shaders**. The project demonstrates real-time graphics programming techniques by combining mathematical heart equations, particle systems, and shader-based visual effects to create a dynamic heartbeat animation.

---

## Features

- Real-time particle-based heart rendering
- Smooth heartbeat animation
- GLSL glow shader
- Ambient particle system
- Object-oriented architecture
- CMake build system
- Cross-platform C++20 implementation

## Build Instructions

Clone the repository:

```bash
git clone https://github.com/ParvazDevRZ/HeartPulse.git
cd HeartPulse
```

Configure the project:

```bash
cmake -S . -B build
```

Build the project:

```bash
cmake --build build
```

Run the executable:

```bash
./build/HeartBeat
```
